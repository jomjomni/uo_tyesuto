# uo_tyesuto

## 目的

- ゲーム開発をしてみる
- WarChestというボードゲームが楽しかったのでそれをアプリで作ってみる

## 開発環境

- [Godot](https://godotengine.org/)で2dゲームとして作成してみる
- 基本的に上記のアプリをインストールすれば作れる
